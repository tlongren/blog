## A Minimal dark theme for Sublime Text 3

Cyanide is a minimal dark theme for Sublime Text 3 that comes with various color options. You can see screenshots of all the different colors in the gallery at the end of this post.

Cyanide can be [installed via Package Control][1] and can also be [found on GitHub][2].

Cyanide includes settings for various tab heights, folder icon sizes, and UI contrast. To enable Cyanide in Sublime Text, open up your user preferences file and replace the theme and color_scheme directives with these:

<pre class="wp-block-preformatted">"color_scheme": "Packages/Theme - Cyanide/Cyanide.tmTheme",
"theme": "Cyanide.sublime-theme"</pre>

I&#8217;m using Cyanide right now and am quite happy with it. I still love [Predawn][3], but it&#8217;s time for a change. So, I&#8217;ll be rolling with the Wood Cyanide theme for a while.

Cyanide [can be found on GitHub][2], along with all the relevant documentation.
