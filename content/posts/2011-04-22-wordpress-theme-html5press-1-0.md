[HTML5Press 1.0][1] is [available for download][2].

The theme was [designed by Jesper][3], all I did was turn it into a WordPress theme.

If you want to know more about HTML5Press, you can read the [announcement post][4], check out [the demo site][5], visit the [HTML5Press page][1], or view the project [on Github][6].

If you have any questions, feel free to ask in the comments on this post or in the comments on [the HTML5Press page][1].

Please report any issues, bugs, or anything broken at the [Github issues page for HTML5Press][7].

You can download [HTML5Press here][2].

Hope you like it!
